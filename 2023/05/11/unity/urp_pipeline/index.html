URP Pipeline

发表于 2023-05-11 更新于 2023-05-12 分类于 Unity

Cameras

URP 管线中有两种类型的照相机：

  • Base Camera - 基本的照相机类型，渲染到屏幕或者纹理上
  • Overlay Camera - 渲染到其他照相机的输出上，可以将一个 Base Camera 的输出和多个 Overlay Cameras 的输出组合起来，这就是 Camera Stacking

Camera culling and rendering order

  1. 获取场景中所有 active 的 Base Cameras，并且分为渲染到屏幕和渲染到纹理这两组
  2. 将所有渲染到纹理的 Base Cameras 按照优先级排序，优先级高的后绘制
  3. 对于每一个渲染到纹理的 Base Camera，Unity 执行下列步骤：
    1. Cull
    2. Render
    3. 对于其每一个 Camera Stack 中的 Overlay Camera，分别执行 Cull 和 Render
  4. 将所有渲染到屏幕的 Base Cameras 按照优先级排序，优先级高的后绘制
  5. 对于每一个渲染到屏幕的 Base Camera，Unity 执行下列步骤：
    1. Cull
    2. Render
    3. 对于其每一个 Camera Stack 中的 Overlay Camera，分别执行 Cull 和 Render
