辐射度量学 ( Radiometry )

辐射度量学提供了一组思想和数学工具来描述光的传播和反射，它是学习 PBR 材质和推导渲染算法的基础。一般情况下，几何光学就足以描述光如何与其他物体相互作用，而当物体对象与光的波长差不多时，就需要使用波动光学来实现一些特殊的现象（色散、干预等）。游戏中大部分情况下使用几何光学就足够了，并且假定满足以下这些条件：

  • 线性：L(A + B) = L(A) + L(B)，A 和 B 表示两个不同的输入光，L 表示给定输入光的情况下的输出
  • 能量守恒：L(A) ≤\le≤ A
  • 无偏振光：只考虑光的波长属性
  • 无荧光或磷光：不同波长的光的行为彼此之间完全独立
  • 稳定状态：环境中的光达到了平衡，辐射分布保持不变

Q=hcλQ = \frac{hc}{\lambda} Q=λhc​

辐照度 / 辐射出射度 ( Irradiance / Radiant Exitance)

E(p)=lim⁡ΔA→0ΔΦ(p)ΔA=dΦ(p)dAE(p) = \lim_{\Delta A \to 0} \frac{\Delta \Phi(p)}{\Delta A} = \frac{d\Phi(p)}{dA} E(p)=ΔA→0lim​ΔAΔΦ(p)​=dAdΦ(p)​



Lambert’s Law

\begin{align} E_1 &= \frac{\Phi}{A_1} = \frac{\Phi}{A} \\ E_2 &= \frac{\Phi}{A_2} = \frac{\Phi}{A/cos\theta} = \frac{\Phi cos\theta}{A} \end{align}

辐射强度 ( Radiant Intensity )

I=lim⁡Δω→0ΔΦΔω=dΦdωI = \lim_{\Delta\omega\to 0} \frac{\Delta\Phi}{\Delta\omega} = \frac{d\Phi}{d\omega} I=Δω→0lim​ΔωΔΦ​=dωdΦ​



辐射率 ( Radiance )

L(p,ω)=lim⁡Δω→0ΔEω(p)Δω=dEω(p)dω=dΦdωdA⊥L(p,\omega) = \lim_{\Delta \omega \to 0}\frac{\Delta E_\omega(p)}{\Delta \omega} = \frac{dE_\omega(p)}{d\omega} = \frac{d\Phi}{d\omega dA^{\perp}} L(p,ω)=Δω→0lim​ΔωΔEω​(p)​=dωdEω​(p)​=dωdA⊥dΦ​

辐射率是非常重要的一个度量单位，这是因为

  1. 知道了辐射率，其他所有的度量单位都可以通过积分计算出来
  2. 光线沿着某一方向传播的过程中辐射率保持不变

光度学

光度学是研究人类视觉系统对于可见光辐射的感知程度，每一个辐射度量单位都有对应的光度度量单位，可以通过对光谱响应曲线 V(λ)V(\lambda)V(λ) 进行积分计算出来

基本度量单位

光能 ( Luminous Energy )

单位是 talbot (T)，对应于辐射能量

光通量 ( Luminous Flux )

单位时间内由光源所发出的总光能，单位 (流明 lm)，对应于辐射通量

照度 / 光出射度 ( Illuminance / Luminous Exitance )

物体表面单位面积所吸收 / 发出的光通量，单位 (勒克斯 lux)，对应于辐照度 / 辐射出射度

发光强度 ( Luminous Intensity )

光源在给定方向上，每单位立体角内所发出的光通量，单位 (坎德拉 cd)，对应于辐射强度

亮度 ( Luminance )

Y=∫λL(λ)V(λ)dλY = \int_\lambda L(\lambda)V(\lambda)d\lambda Y=∫λ​L(λ)V(λ)dλ

单位面积光源在给定方向上，每单位立体角内所发出的光通量，单位 (尼特 nit)，对应于辐射率
