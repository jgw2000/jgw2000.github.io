漫反射

Lambertian 漫反射模型

这是最简单的 BRDF 模型之一，描述了一个完美的漫反射表面，入射光经过该表面向各个方向反射的辐射度都相同。反射率 albedo 定义如下

ρalbedo=∫H2(n)fr(p,ωo,ωi)  cosθi  dωi \rho_{albedo} = \int_{H^2(n)} f_r(p,\omega_o,\omega_i) \; cos\theta_i \; d\omega_i ρalbedo​=∫H2(n)​fr​(p,ωo​,ωi​)cosθi​dωi​

其表示半球上的常量入射光经过物体表面反射到某一给定方向 ωo\omega_oωo​ 上的比例，又根据 BRDF 的可逆性，也等价于表示某一给定方向的入射光经过物体表面反射到整个半球面上的比例

又因为 fr(p,ωo,ωi)f_r(p,\omega_o,\omega_i)fr​(p,ωo​,ωi​) 是常量，可得

fr(p,ωo,ωi)=ρalbedoπ f_r(p,\omega_o,\omega_i) = \frac{\rho_{albedo}}{\pi} fr​(p,ωo​,ωi​)=πρalbedo​​

Point2f SampleUniformDiskConcentric(Point2f u) {
    // 将 u 映射到 [-1, 1]x[-1, 1] 并且处理原点处的退化
    Point2f uOffset = 2 * u - Vector2f(1, 1);
    if (uOffset.x == 0 && uOffset.y == 0)
        return {0, 0};
    
    // 应用同心映射
    Float theta, r;
    if (std::abs(uOffset.x) > std::abs(uOffset.y)) {
        r = uOffset.x;
        theta = PiOver4 * (uOffset.y / uOffset.x);
    } else {
        r = uOffset.y;
        theta = PiOver2 - PiOver4 * (uOffset.x / uOffset.y);
    }
    return r * Point2f(std::cos(theta), std::sin(theta));
}

由于

p(r,θ,ϕ)=r2sinθ  p(x,y,z) p(r,\theta,\phi) = r^2 sin\theta \; p(x,y,z) p(r,θ,ϕ)=r2sinθp(x,y,z)

余弦加权采样半球面 Cosine-Weighted

许多光传输积分公式包含一个余弦因子，根据重要性采样性质，有一种根据半球面的余弦加权分布生成方向的方法会很有用，该方法会更有可能采样得到接近球顶附近的方向。从数学上来讲，这意味着

p(ω)∝cosθ p(\omega) \propto cos\theta p(ω)∝cosθ

进行归一化

∫H2p(ω)dω=1∫02π∫0π2c cosθ sinθ dθ dϕ=1c 2π∫0π/2cosθ sinθ dθ=1c=1π \begin{aligned} \int_H^2 p(\omega) d\omega &= 1 \\ \int_0^{2\pi}\int_0^{\frac{\pi}{2}} c\,cos\theta\,sin\theta\,d\theta\,d\phi &= 1 \\ c\,2\pi\int_0^{\pi/2} cos\theta\,sin\theta\,d\theta &= 1 \\ c &= \frac{1}{\pi} \end{aligned} ∫H2​p(ω)dω∫02π​∫02π​​ccosθsinθdθdϕc2π∫0π/2​cosθsinθdθc​=1=1=1=π1​​

因此

p(ω)=1πcosθ p(\omega) = \frac{1}{\pi} cos\theta p(ω)=π1​cosθ

我们可以是使用 Malley 方法来采样该分布

Malley 方法

从单位圆中均匀采样点，然后将该点向上投影到半球面上，如此获得的采样点即满足余弦加权分布

镜面反射和透射

折射率

定义

通常用折射率 IOR 来概括光的速度降低，例如，光遇到折射率等于 2 的材质表面后速度会降为原来的一半。折射率的范围通常在 1.0 - 2.5 之间且随光的波长而变化，用希腊字母 η\etaη 表示。

当光遇到 IOR 值突然变化的边界时会发生明显的反射，折射率 η\etaη 控制了表面的外观表现，因此对于该值的良好估计对于基于物理的渲染非常重要。

镜面反射定律

给定入射光的方向 (θi,ϕi)(\theta_i,\phi_i)(θi​,ϕi​)，完美镜面的反射方向 (θr,ϕr)(\theta_r,\phi_r)(θr​,ϕr​) 满足以下性质：

θr=θiϕr=ϕi+π \begin{aligned} \theta_r &= \theta_i \\ \phi_r &= \phi_i + \pi \end{aligned} θr​ϕr​​=θi​=ϕi​+π​

斯涅尔定律 Snell's Law

定义

斯涅尔定律描述了光的入射方向 (θi,ϕi)(\theta_i,\phi_i)(θi​,ϕi​) 和透射方向 (θt,ϕt)(\theta_t,\phi_t)(θt​,ϕt​) 之间的关系，依赖于边界两边介质的折射率 ηi\eta_iηi​ 和 ηt\eta_tηt​，满足如下性质：

ηi sinθi=ηt sinθtϕt=ϕi+π \begin{aligned} \eta_i \, sin\theta_i &= \eta_t \, sin\theta_t \\ \phi_t &= \phi_i + \pi \end{aligned} ηi​sinθi​ϕt​​=ηt​sinθt​=ϕi​+π​

一个有用的观察：斯涅尔定律不依赖于具体的 ηi\eta_iηi​ 和 ηt\eta_tηt​ 的值，而是他们的比值，因此该定律也可以写成

sinθi=η sinθt sin\theta_i = \eta \, sin\theta_t sinθi​=ηsinθt​

其中 η=ηtηi\eta = \frac{\eta_t}{\eta_i}η=ηi​ηt​​ 表示相对折射率

提示

导体的折射率 IOR 是一个复数，其中实数部分和之前一样描述光的速度降低，而虚数部分描述光在导体内部传播过程中的衰减。这个衰减非常快，对反射的多少也有显著影响，因此我们也必须考虑这个因素。

导体 BRDF 基于两个物理概念：镜面反射定律决定每条光线的反射方向，以及菲涅尔等式决定反射光的比例。剩余的光会折射到导体中，被快速吸收并转化为热量。

令 Fr(ω)F_r(\omega)Fr​(ω) 表示来自 ω\omegaω 方向光的菲涅尔反射比例，我们需要求解 BRDF frf_rfr​ 使得满足以下等式：

Lo(ωo)=∫H2(n)fr(ωo,ωi)Li(ωi)∣cosθi∣dωi=Fr(ωr)Li(ωr) L_o(\omega_o) = \int_{H^2(n)} f_r(\omega_o,\omega_i) L_i(\omega_i)|cos\theta_i|d\omega_i = F_r(\omega_r)L_i(\omega_r) Lo​(ωo​)=∫H2(n)​fr​(ωo​,ωi​)Li​(ωi​)∣cosθi​∣dωi​=Fr​(ωr​)Li​(ωr​)

根据 Dirac delta 分布性质

∫f(x)δ(x−x0)dx=f(x0) \int f(x)\delta(x-x_0) dx = f(x_0) ∫f(x)δ(x−x0​)dx=f(x0​)

可以猜测导体 BRDF 等于

fr(ωo,ωi)=δ(ωi−ωr)Fr(ωi) f_r(\omega_o,\omega_i) = \delta(\omega_i - \omega_r) F_r(\omega_i) fr​(ωo​,ωi​)=δ(ωi​−ωr​)Fr​(ωi​)

代入可得

Lo(ωo)=∫H2(n)δ(ωi−ωr)Fr(ωi)Li(ωi)∣cosθi∣dωi=Fr(ωr)Li(ωr)∣cosθr∣ \begin{aligned} L_o(\omega_o) &= \int_{H^2(n)} \delta(\omega_i-\omega_r) F_r(\omega_i) L_i(\omega_i) |cos\theta_i| d\omega_i \\ &= F_r(\omega_r)L_i(\omega_r)|cos\theta_r| \end{aligned} Lo​(ωo​)​=∫H2(n)​δ(ωi​−ωr​)Fr​(ωi​)Li​(ωi​)∣cosθi​∣dωi​=Fr​(ωr​)Li​(ωr​)∣cosθr​∣​
